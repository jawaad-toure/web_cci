# Projet PHP de l'UE Langages du Web du Master CCI (Laravel 10)

- Installation des dépendances du projet :
    ~~~
    composer install
    ~~~

- Création et alimentation de la base de données :
    ~~~
    php artisan db:seed
    ~~~

- Démarrage du serveur HTTP :
    ~~~
    php artisan serve
    ~~~

- Exécution des tests :
    ~~~
    php artisan test
    ~~~
